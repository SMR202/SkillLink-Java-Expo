Senior Frontend Developer for SaaS Dashboard Revamp

code Web Development schedule Posted 2 hours ago payments Est. Budget: $5,000 - $8,000

Job Description

We are a growing B2B SaaS company looking for an experienced frontend developer to overhaul our main analytics dashboard. The current interface is built on legacy React components and needs to be modernized using Next.js, Tailwind CSS, and a more robust state management solution.

Key Requirements:

  • Proven experience with React, Next.js, and modern CSS frameworks (Tailwind preferred).
  • Strong understanding of responsive design and building accessible UI components.
  • Experience integrating RESTful APIs and managing complex client-side state.
  • Ability to translate Figma designs into pixel-perfect, interactive frontend code.

Project Scope:

The project involves rebuilding 5 core dashboard views, implementing new data visualization charts (using Recharts or similar), and optimizing load performance. We expect clean, modular, and well-documented code.
